# 📞 Agenda Telefónica Profesional

Este proyecto es una aplicación de escritorio desarrollada en Java usando **Swing**, que permite gestionar una **agenda de contactos** de forma visual, intuitiva y funcional. Fue creada como parte de un **Hackathon técnico**, cumpliendo con todos los requerimientos funcionales establecidos.

---

## 🚀 Funcionalidades

- ➕ Añadir nuevos contactos (nombre, apellido y teléfono)
- 🔍 Buscar contacto por nombre, apellido o ambos
- 📋 Listar todos los contactos ordenados alfabéticamente
- 🗑️ Eliminar contactos desde una lista desplegable
- ✏️ Modificar el número de teléfono de un contacto existente
- ❗ Verificar si la agenda está llena
- 🔢 Consultar cuántos espacios quedan disponibles

---

## ✅ Requisitos cumplidos

- [x] Validación de campos vacíos (nombre y apellido no pueden estar vacíos)
- [x] Evita contactos duplicados (nombre + apellido iguales sin importar mayúsculas)
- [x] Agenda con tamaño configurable y por defecto (10)
- [x] Todos los métodos requeridos implementados y conectados a la interfaz gráfica
- [x] Mensajes claros para el usuario final

---

## 🛠️ Tecnologías utilizadas

- Java 17+
- Swing (GUI nativa de Java)
- IntelliJ IDEA (o cualquier IDE compatible con Java)
- Git y GitHub para control de versiones
